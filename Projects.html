Dsa-Wizard

Project link Github link
The DSA-Bot is an interactive tool designed to teach Data Structures and Algorithms (DSA) using the Socratic method. Instead of giving direct answers, it asks thought-provoking questions to help users develop a deeper understanding of DSA concepts. This bot offers an engaging, question-driven learning experience for students and developers aiming to master DSA efficiently.
